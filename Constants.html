  • kGREYConfigKeyCALayerModifyAnimations

    Configuration that determines whether or not CALayer animations are modified. If @c YES, then cyclic animations are set to only run once, and animation duration is limited to a maximum of @c kGREYConfigKeyCALayerMaxAnimationDuration. @remark This should only be used if synchronization is disabled, since otherwise cyclic animations will cause EarlGrey to timeout and fail tests.

    Accepted values: @c BOOL (@c YES or @c NO) Default value: @c YES

  • kGREYConfigKeyURLBlacklistRegex

    Configuration that holds a regular expression used to determine whether EarlGrey should synchronize with a web request made to a URL or not. EarlGrey will not synchronize with requests made to URLs matching the configured regular expression.

    Accepted values: @c nil or any regex strings accepted by NSRegularExpression. The empty string and @c nil indicate that EarlGrey must synchronize with all network URLs. Default value: @c nil (synchronize with all network URLs.).
