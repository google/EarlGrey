    Declaration

    Objective-C

    + (id<GREYAction>)actionForMoveSliderToValue:(float)value;

    Parameters

    The value to which the slider should be moved. If this is not attainable after a reasonable number of attempts (currently 10) we assume that the @c is unattainable for a user (it is probably the case this value resides between two pixels).
